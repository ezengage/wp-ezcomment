ezComment 第三方评论平台, 让用户使用新浪微博，腾讯微博，人人网, 网易微博，搜狐微博等帐号登录发表评论。

== Description ==

[ezComment](http://ezcomment.ezengage.com/ "ezComment") 提供评论平台服务。


== Installation ==

要求 Wordpress 3.0 及以上版本

1. 复制 'ezComment' 目录和其中的内容到你的 `/wp-content/plugins/` 目录 或者在Wordpress 后台安装。

2. 通过'插件'('Plugins') 菜单激活 ezComment 插件。

3. 访问'设置'('Settings') -> 'ezComment' 来访问 ezComment 配置页面。
